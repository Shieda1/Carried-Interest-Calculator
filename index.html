What is the carried interest?
The carried interest is the share of the profit that should be paid to the portfolio manager.
How to calculate carried interest?
To calculate carried interest you should use the following formula:
carry distribution = (final fund value - initial fund value * (1 + (hurdle rate / 100))) * (carried interest / 100)
